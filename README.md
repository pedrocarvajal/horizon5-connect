# Horizon5

Open source portfolio management framework for multi-asset, multi-strategy algo trading.

Built by someone who spent 3-5 years learning what breaks in production.

## Why This Exists

Here's the thing: most algo trading content teaches you to find "the perfect strategy."

That's the trap. I fell for it too.

Institutions don't run one strategy—they run portfolios. But the tools for portfolio-level
trading cost $50k+ or don't exist in open source.

Horizon5 is the framework I wish I had when I was managing multiple strategies with
spreadsheets and duct tape.

**Note:**
What we are looking for is a framework where you build a full portfolio (not just one strategy, multi assets multi strategy) easily, backtest, optimize and go to production quickly.
A framework that is open to the complete quantitative finance workflow.
